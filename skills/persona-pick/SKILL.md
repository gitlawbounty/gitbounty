# persona-pick

Get the current weekly bounty picks from a gitbounty AI persona. Each persona has a distinct specialization and selection style. All picks generated by Groq Llama 3.3 70B against the live bounty list.

## Personas

| Persona | Glyph | Specialty | Style |
|---|---|---|---|
| **ORACLE** | ◆ | Research, data, indexing | Methodical signal-hunter, prefers data/research bounties with clear deliverables |
| **CIRCUIT** | ▲ | Infra, contracts, deploys | Infra-first, prefers contract/devops/deployment work |
| **AURORA** | ✦ | Creative, frontend, UX | Aesthetics-driven, prefers frontend/design/UX bounties |
| **WAGER** | ◈ | High payout, tight deadlines | Risk-on, picks long-tail high-reward bounties with steep difficulty |

## Inputs

| Field | Type | Description |
|---|---|---|
| `persona` | `'oracle' \| 'circuit' \| 'aurora' \| 'wager'` | Which persona's picks to fetch |
| `week` | `string` (optional, ISO date) | Which week's picks. Default: current week |

## Outputs

```jsonc
{
  "persona": "oracle",
  "week": "2026-W20",
  "picks": [
    {
      "bountyId": "9898bec6-1b0c-4980-96f6-a0220b00fec6",
      "rank": 1,
      "reasoning": "Spec is unambiguous, deliverable is a single artifact, escrow is set. Low-friction win.",
      "confidence": 0.88
    }
  ],
  "commentary": "Light week on research-oriented bounties. Pick tight; quality over quantity.",
  "generatedAt": "ISO timestamp"
}
```

## Invocation

```bash
curl https://gitlawbounty.vercel.app/api/persona/oracle/picks
curl https://gitlawbounty.vercel.app/api/persona/circuit/picks
curl https://gitlawbounty.vercel.app/api/persona/aurora/picks
curl https://gitlawbounty.vercel.app/api/persona/wager/picks
```

## Implementation

Weekly Vercel Cron calls each persona's LLM with:
1. Current open bounties from the gitlawb network firehose (`node.gitlawb.com/api/v1/bounties`)
2. The persona's system prompt (defines specialty + selection style)
3. Cached past picks for history-aware curation

LLM returns ranked picks with reasoning. Edge-cached for the week.
